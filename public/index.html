Who I Am

Coder

I love code. I love the challenge it gives me as I work on projects, and solve each one's unique problems. From styling a web site's look with the lightweight Bourbon library, to the JavaScript that powers the user experience up front and fetches data from the server, I aim to build web sites and web apps with a consistent feel and thoughtful thoroughness.

Creator

Creating things is as natural as breathing. Ideas arrive constantly and give me so much to work out and put to good use. Ideas as simple as a task list, to something as grand as open-source inventory tracking software, I continually give myself something worthwhile to work toward.

Explorer

Few things excite me more than "new". Whether it is a new place- such as my first-ever trip to southern California recently, or discovering a new database (RethinkDB) that fits closely with my belief that the web should be as real-time as possible, I never stop exploring places or ideas.

Projects



Task List

This project was my final project at Thinkful. It is a full-stack app with a React/Redux front-end and, using Node.js and Socket.io, connects to a RethinkDB database to keep track of tasks via a persistent list. The tech stack that I used here is one that I am interested in using as my main stack for any future side-projects as it lends itself very nicely to the real-time web.



Hot or Cold

Although initially built as pure HTML, CSS, and JavaScript, I ended up refactoring this when I started learning React and Redux. I created the components necessary and set up the flow of info through the app. It was hard initially to wrap my head around the abstraction of React, but I was able to gain a good understanding of it. Also, the addition of Redux really helped to make things easier for me.



API "Hack"

In this project, I learned how to interact with the endpoints of an API. As it was the introduction of basic CRUD functions, it actually dovetailed nicely into my rising interest in Node.js. This one is just a simple recommendation program for any books, shows, movies, and more that you enjoy.



Flappy Bird

A project that taught me a lot of JavaScript, Flappy Bird was my first real introduction to that language. I learned how to link JavaScript files together and keeping the code simple to read and understand. This also introduced me to Node.js (not used in this project though) by showing me require statements to link the files together. This sparked an interest in Node.js that lead me to enlarging my skillset and becoming a Full-Stack JavaScript Developer.
